# git

### git 설치확인

```
git
```  

### git 환경 설정하기
```
git config --global user.name `your name`
git config --global user.email `your email`
```

### git 초기화
```
mkdir new-project
cd new-project
git init
```

### git 상태확인
```
git status
```

### git 스테이징 하기
```
git add filename

// 전체
git add .
```

### git 커밋하기
```
git commit -m "commit message"
```
